Sobre o Curso

Este site foi criado para praticar os principais elementos do HTML: titulos, paragrafos, listas, links, imagens, tabelas, e formularios.

O objetivo é entender a estrutura básica de um documento HTML, sem usar CSS nem JavaScript

Conteúdo das Aulas

  1. introdução à web e o HTML
  2. estrutura básica de uma página
